- `logo.png` (400x400) - 高清logo

请确保所有图像都经过优化，文件大小适中，并包含相关的alt文本。

图像应该：
1. 使用现代格式 (WebP, AVIF)
2. 包含适当的压缩
3. 具有响应式尺寸
4. 支持深色/浅色模式变体
